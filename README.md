## Input: A list of numbers (e.g., [8, 3, 1, 5]) is entered by the user, The user must then run the program, The program Processes the mergesort function recursively splits the list until all sub-lists are size 1. Then the program processes the merge function compares and combines these sub-lists in sorted order. The program then outputs the fully sorted list (e.g., [1, 3, 5, 8]) and, crucially, a demonstration of the steps (e.g., a visualization or a text log of the splits and merges).
